## Requirements

01) Java 17
02) Maven 3.8.8
03) Spring Boot 3.3.11
04) Spring Web
05) Spring OpenFeign

## Project setup

01) Clone the project

    	git clone https://github.com/awokou/spring-boot-feign-client.git

02) Clean and build the project using maven

    	- mvn clean install
        - mvn spring-boot:run
